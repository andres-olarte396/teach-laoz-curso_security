# Guion de Audio - 5.1.1 Defensa Perimetral

**Locutor**: "Módulo 5. Vamos a salir del código un momento para mirar la infraestructura."

**Locutor**: "Puedes tener el código más seguro del mundo, pero si tu base de datos está expuesta a internet con el usuario 'sa' y password 'admin', estás perdido. La seguridad es capas. **Locutor**: "Tu trabajo como desarrollador .NET es saber qué puertas necesitas abrir para que tu app funcione, y exigir que TODAS las demás estén cerradas."
