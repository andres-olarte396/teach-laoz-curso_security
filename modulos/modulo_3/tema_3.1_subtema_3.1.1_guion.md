# Guion de Audio - 3.1.1 Autenticación Moderna

**Locutor**: "Bienvenidos al Módulo 3. Vamos a hablar de Identidad."

**Locutor**: "En el pasado, hacíamos un `if (user == admin)`. Hoy, eso no escala. Hoy hablamos de Tokens JWT y Claims."

**Locutor**: "Imagina un JWT como un pasaporte. El pasaporte dice quién eres (Subject), qué país te lo dio (Issuer) y cuándo vence (Expiration). La aduana (tu API) confía en el pasaporte porque tiene un sello holográfico (Firma) difícil de falsificar. No llaman al presidente de tu país cada vez que cruzas la frontera."

**Locutor**: "Así funcionan los JWT. Permiten que tu API sepa quién es el usuario rápidamente, sin consultar la base de datos a cada milisegundo. Eso es velocidad y seguridad."
